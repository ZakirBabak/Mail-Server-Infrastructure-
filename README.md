
# Mail Server Infrastructure

## Project Overview
This project demonstrates the deployment of a secure email server infrastructure using Linux and open-source technologies.

It simulates a real enterprise mail system including SMTP, IMAP, webmail access, authentication and DNS configuration.

---

## Technologies Used

- Ubuntu Server / Linux
- Postfix (SMTP server)
- Dovecot (IMAP server)
- Mailcow (Docker-based mail platform)
- **SOGo (Webmail & Collaboration Suite)**
- DNS configuration (Cloudflare)
- SSL/TLS encryption (Let's Encrypt)
- Docker & Docker Compose

---

## Objectives

- Deploy a fully functional mail server infrastructure
- Configure secure email communication (SMTP / IMAP)
- Provide webmail access via SOGo
- Implement SSL/TLS encryption
- Manage domain-based email system
- Ensure secure authentication and email delivery

---

## Security Features

- TLS encryption for email communication
- Secure authentication system
- Spam and relay protection
- Firewall and access control configuration

---

## Features Implemented

- SMTP server configuration (Postfix)
- IMAP service configuration (Dovecot)
- Webmail interface (SOGo / Mailcow)
- DNS records setup (MX, SPF, DKIM, DMARC)
- SSL certificate deployment (Let's Encrypt)
- Email sending and receiving tests
